🟡 ปุ่มขาวเหลือง = กด · 🟠 ปุ่มดำส้ม = กด (มีปุ่มดำแค่ B♭ ตัวเดียวทั้งเพลง) · ทุกคอร์ดกด 3 นิ้วพร้อมกัน (นิ้วโป้ง-กลาง-ก้อย)

🗺️ โครงเพลง + ลำดับคอร์ด (เล่นวนตามนี้)

Intro / Verse F Dm F Dm (วนสลับ F ↔ Dm ท่อนละ 1 ห้อง)
Pre-chorus B♭ C Dm F
Chorus B♭ C F F  →  B♭ C Dm F

🎵 ทำนองหลัก (มือขวา · เวอร์ชัน simplified)

ACCD   AGFED
BFFA   AA
CB♭AGFED   A   D

อ่านซ้ายไปขวา · แต่ละตัว = 1 จังหวะ (เว้นวรรค = พักนิด) · ตัวอักษร = ชื่อโน้ตปุ่มขาว ยกเว้น B♭ = ปุ่มดำ · เล่นในช่วงกลางคีย์บอร์ด (รอบ ๆ middle C)

👐 วิธีเล่นแบบง่ายสุด (2 สเต็ป)

① เริ่มจากคอร์ดล้วน (ง่ายสุด)

  • มือซ้าย: กดโน้ตตัวเดียว = ตัวแรกของคอร์ด (F→กด F, Dm→กด D, B♭→กด B♭, C→กด C)
  • มือขวา: กดคอร์ดเต็ม 3 นิ้ว ตามไดอะแกรมด้านบน
  • เปลี่ยนคอร์ดตามลำดับ "โครงเพลง" · ร้อง/ฮัมทำนองตามไป

② เพิ่มทำนอง (ขั้นถัดไป)

  • มือขวา: เล่นทำนองหลัก (ช่อง 🎵 ด้านบน) แทนการกดคอร์ด
  • มือซ้าย: กดโน้ตเบส (ตัวแรกของคอร์ด) ค้างไว้ทั้งห้อง
  • พอคล่อง ค่อยสลับ คอร์ด ↔ ทำนอง ให้เพลงเต็มขึ้น

💡 เคล็ดลับมือใหม่

  • ซ้อมช้าก่อน — ตั้งจังหวะ ~68 BPM (ครึ่งความเร็วจริง) แล้วค่อยเร่งเป็น 136
  • ฝึกเปลี่ยนคอร์ดคู่ที่เจอบ่อยสุดก่อน: F ↔ Dm (ต่างกันแค่ขยับนิ้วนิดเดียว) แล้วค่อยเพิ่ม B♭, C
  • นับ 1-2-3-4 ทุกห้อง เปลี่ยนคอร์ดที่จังหวะ "1"
  • ทั้งเพลงมีแค่ 4 คอร์ด วนซ้ำ — จำ 4 รูปนี้ได้ = เล่นได้ทั้งเพลง
  • อยากง่ายกว่านี้อีก บอกได้ — แปลงเป็น คีย์ C major (ปุ่มขาวล้วน ไม่มีปุ่มดำเลย) ให้ได้
